📊 Analisis Perhitungan Persentase di Excel
Deskripsi
Proyek ini berfokus pada pengolahan berbagai jenis perhitungan berbasis persentase menggunakan Microsoft Excel, seperti diskon, kenaikan harga, laba, pertumbuhan, dan proporsi data.

Dataset
Data terdiri dari beberapa kasus:
- Perhitungan persentase dari suatu angka
- Perhitungan diskon dan harga setelah diskon
- Perhitungan kenaikan harga
- Perhitungan laba dan persentase keuntungan
- Perhitungan pertumbuhan data per tahun
- Perhitungan persentase terhadap total

Tools & Fungsi yang Digunakan
- Microsoft Excel
- Operasi aritmatika dasar
- Perhitungan persentase (%)
- Analisis data sederhana

Hasil Analisis
- Diskon dan kenaikan harga dapat dihitung secara otomatis
- Persentase laba dan pertumbuhan diperoleh dengan akurat
- Proporsi data terhadap total dapat diketahui
- Data tersusun secara terstruktur dan mudah dipahami

Tujuan Proyek
- Melatih pemahaman konsep persentase
- Menerapkan persentase dalam berbagai kasus
- Mengolah data menjadi informasi yang lebih jelas

Insight
Konsep persentase dapat digunakan dalam berbagai konteks, mulai dari keuangan hingga analisis data, sehingga menjadi dasar penting dalam pengolahan data.

Penutup
Proyek ini merupakan latihan pengolahan data berbasis persentase dengan berbagai penerapan dalam kehidupan sehari-hari.
